1. What is HTTP?

HTTP is ultiately the communication system between the client and the server. Whenever a user wants to access a webpage a requestion is sent to the server who is hosting the site. Then, the server will send back a response which usually displays the webpoge for the user.

2. What is HTML?

3. Describe the process of requesting a web page from a web server.

Step 1: You tyoe a URL or click a link then the browser resvolves the host name to an IP address
Step 2: Browser will then open a connection and sends an HTTP request.
Step 3: The server process the request and sends back an HTTP response woth a status code and HTML file.
Step 4: The browser then analyzes the HTML and requests any additional files it references like images. Finally, providing the page for you.

4. What is the difference between a regular tag and an empty tag?

A regular tag has both an opening and a closing tag with content in between. An empty tag has no closing tag and no content between its oping and closing form; forming the function of creating a line break or inserting an image.
